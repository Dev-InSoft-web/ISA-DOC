Cálculo personalizado de datos de la operación

Algunas columnas del listado de productos de la operación pueden calcularse automáticamente usando una expresión definida por el usuario. Una expresión de cálculo en [CONTAPYME], se puede definir como una cadena de texto formada por una o varias instrucciones, que al ser evaluadas generan un único dato resultante. Ejemplo:

Cantidad de cajas = Cantidad de producto / 10.

La característica principal de estas expresiones es que pueden combinarse para los cálculos, datos de la operación, de los productos y del tercero. Además, se cuenta con una gran variedad de operadores, instrucciones y funciones de lenguajes de programación como Pascal, JavaScript, C++ y Visual Basic, así como las funciones de ContaExcel.

Para mayor información sobre las funciones de ContaExcel y cómo usarlas, ingrese a [Editor de expresiones para el cálculo > Menú > Ayuda > Funciones ContaExcel]

Al activar este botón, indica al sistema que el resultado de la expresión es un valor alfanumérico. Si no está presionado, el resultado se trata como un valor numérico. Además, todos los valores de las variables que se usen dentro de la expresión tendrán como valor por defecto el cero (0) cuando éstos estén vacíos o sin asignar.
