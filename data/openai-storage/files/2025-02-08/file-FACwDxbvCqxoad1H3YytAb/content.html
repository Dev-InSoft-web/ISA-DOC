Cód. Activo

Código que identifica al activo que se pretende dar de baja.
Ejemplo
Se tiene un catálogo con el siguiente listado de activos:
Activo Código Activo Nombre
VEH001 Vehículo Ford KII-735
COM001 Computador Dell 1580
COM002 Computador portátil Acer 1025
Se pretende dar de baja (por robo) el computador Dell 1580, por lo tanto en el campo "Código", se registra: COM001.
