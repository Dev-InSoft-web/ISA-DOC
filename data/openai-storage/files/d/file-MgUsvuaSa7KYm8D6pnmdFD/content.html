Adiciona una actividad

Esta opción permite importar actividades empresariales con su correspondiente conjunto de labores (tareas) asociadas.

Si da clic, se abrirá el selector de actividades, deberá seleccionar la actividad a importar, dar clic en "Seleccionar", luego deberá asignar el código con el cual quiere que se identifique la actividad a importar (el sistema sugiere un código por defecto), finalmente la actividad se adiciona al listado de actividades.

Las actividades y labores son muy útiles cuando se manejan costos de producción y/o cuando se quiere costear, causar y pagar la mano de obra por labor realizada o jornales trabajados (importante, este manejo no se debe confundir con el módulo de nómina contable).

Observaciones
El sistema automáticamente incluye la actividad cero, la cual no se puede eliminar ni recodificar.

Cada vez que se importa una actividad, se importan también las labores relacionadas para esa actividad.

Las actividades adicionadas previamente, se muestran en fondo de solo lectura y no se pueden remover de la lista; si de sea remover una actividad deberá ir al catálogo de actividades y eliminarla.

Configuración
Para abrir el catálogo de actividades empresariales, ver [Cinta de opciones: Costos > Catálogos > Actividades empresariales].

Para abrir el catálogo de labores, ver [Cinta de opciones: Costos > Catálogos > Labores], deberá seleccionar la actividad a la cual le desea consultar las labores.
